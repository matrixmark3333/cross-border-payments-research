全球企业每年跨境转移约$23.5万亿（约占全球GDP的25%）；交易成本超$1200亿/年（不含外汇成本）；隐性成本（流动性积压+延迟结算）另行叠加。Oliver Wyman / J.P. Morgan, 2021
$23.5T/年的企业跨境支付规模是全球最大的单一支付市场细分，而$1200亿/年的显性成本仅是冰山一角——流动性积压（预付款项锁定）和延迟结算的隐性成本可能同等量级，这是mCBDC等解决方案能够释放的真正价值。

全规模mCBDC网络可将跨境交易收入（成本）降低约80%（约$1000亿/年，不含外汇收入）。Oliver Wyman / J.P. Morgan mCBDC报告

当前跨境支付痛点三维：成本高（层层中间行手续费）、速度慢（2-5工作日）、透明度低（追踪困难）；而跨国可扩展无缝方案至今仍难实现。Oliver Wyman / J.P. Morgan
三大痛点（成本/速度/透明度）是跨境支付行业创新的核心驱动力，也是Wise（透明度+低成本）、Airwallex（速度+低成本）、SWIFT GPI（透明度+速度）等不同解决方案的差异化切入点。这三个维度的同步解决需要基础设施级别的变革。

1.市场总览
全球跨国公司(MNC)财务总监语录："mCBDC可将现金管理提升至新水平——可及性（随时存取流动性）、可兑换性（随意换汇）、可达性（超越银行账户）、可追溯性（资金流清晰）"。全球科技MNC财务总监访谈，Oliver Wyman
来自实际财务负责人的四个"可"（Accessibility/Convertibility/Reachability/Traceability）精准定义了企业级跨境支付的需求层次，为跨境支付产品设计提供了直接来自买方的需求框架，比分析师的推测更有参考价值。

3.商业模式
私人机构、商业银行和央行近年已推出多项跨境支付改善倡议，但"能跨国家/货币/支付系统的可扩展无缝方案"仍然难以实现。Oliver Wyman / J.P. Morgan, 2021
这一现状观察（尽管2021年发布，但至2025年仍基本成立）说明碎片化是跨境支付的根本性问题，而非技术问题——不同国家的监管、语言、账户体系阻止了"通用解决方案"的出现。这为多轨道策略（multi-rail）的持续价值提供了依据。

🔗 mCBDC网络设计与实施 5
mCBDC报告概述四大设计要素：①数据与隐私；②技术互操作性；③功能（含信贷扩展）；④治理框架（央行/商业银行/服务商角色）。Oliver Wyman / J.P. Morgan
这四个设计维度是mCBDC（及mBridge类实验）的核心议题，至今仍是央行合作的主要分歧所在。治理框架（特别是参与方角色分配）是mBridge项目遇到挫折的主要原因，说明技术问题已解决但政治/治理问题仍是瓶颈。

5.基础设施
mCBDC的核心建设模块：CBDC铸造与赎回 → 外汇转换 → 跨境结算 → 参与方：央行+商业银行+服务商+流动性提供商。Oliver Wyman / J.P. Morgan
mCBDC的四步流程（铸造→FX→结算→分发）与私营跨境支付的流程高度相似，差异在于央行信用背书替代了私营平台信用。商业银行在mCBDC中仍扮演关键中介角色，并非被disintermediated，这解释了银行支持mCBDC试点的动机。

5.基础设施
mCBDC对传统代理行模式的挑战：削减中间行层次，直接实现多国货币跨境实时结算；但也为现有参与者提供新能力扩展机会（而非替代）。Oliver Wyman / J.P. Morgan
"挑战+机会并存"的论述是此类报告（由商业机构J.P. Morgan参与撰写）的常见立场——强调机会以降低利益相关方的抵制。但实质上，mCBDC若成功将显著压缩代理行（含J.P. Morgan）的手续费收入，这是不可回避的利益冲突。

5.基础设施
mCBDC对不同参与方的分层影响：央行（货币主权+新政策工具）、商业银行（新服务能力+中介角色保留）、科技/第三方服务商（新基础设施商机）。Oliver Wyman / J.P. Morgan
分层影响框架揭示了mCBDC生态中不同类型公司的战略布局方向：传统银行宜成为mCBDC的核心参与机构（维护关系），而fintech公司宜布局"第三方服务商"角色（基础设施整合者），这是Airwallex/Wise等平台在mCBDC时代的潜在定位。
