7.区域5.基础设施
2019年底CIPS有33家直接参与者和903家间接参与者（覆盖94国），较2015年分别增长74%和413%；通过网络触达167个国家3000+银行。Wikipedia / CIPS 2019年数据
间接参与者413%增速说明接入门槛（依托直接参与行）有效降低，网络外部性显著放大。

5.基础设施
CIPS超过80%的交易依赖SWIFT报文网络，同时采用ISO 20022标准，支持中文字符。Wikipedia / CIPS架构
依赖SWIFT报文是CIPS的结构性短板——美国若制裁CIPS报文传输，将产生影响。但直接参与行可绕过SWIFT专线接入，是重要的冗余设计。

CIPS与人民币SDR纳入同步推进，属于国家战略级基础设施，政策与商业双轮驱动是其增长的核心逻辑。

6.监管5.基础设施
2018年3月CIPS二期上线，实现DVP（券款对付）结算，支持北向债券通；5月全面运营，扩大直接参与行范围。Wikipedia / CIPS Phase 2
DVP结算降低结算风险，支持债券通说明CIPS从贸易支付延伸至金融市场，覆盖场景大幅扩展。

5.基础设施3.商业模式
2016年3月SWIFT与CIPS签署MOU，SWIFT进行ISO 20022实施，支持中文字符；2017年9月SWIFT通过SWIFTRef发布CIPS参与行参考数据，含BIC、LEI、IBAN等。Wikipedia / SWIFT-CIPS合作
SWIFT与CIPS形成合作-竞争关系，SWIFT接入CIPS有助于扩大海外间接参与者，短期共生但长期存在替代张力。

4.竞争5.基础设施
CIPS由人民银行批准，上海总部，公司制企业法人（合作社性质），负责跨境人民币清算结算，同时支持离岸人民币业务。Wikipedia / CIPS概述
CIPS是人民银行主导的国家战略级基础设施，政策连续性强，但也意味着商业化扩张受政策约束。
央行监管国家战略

6.监管5.基础设施
CIPS订阅并维护多项ISO标准的注册机构（RA）资质，包括ISO 9362（BIC）、ISO 10383（MIC）、ISO 13616（IBAN）、ISO 15022、ISO 20022。Wikipedia / ISO标准
掌握多个ISO标准RA资质，使CIPS在国际金融标准体系中具有制度性话语权，为人民币国际化提供标准支撑。

6.监管5.基础设施
一带一路沿线59个国家和地区（含港澳台）的1017家银行于2019年底通过CIPS开展业务。Wikipedia / 一带一路数据
一带一路是CIPS扩张的主要地理轴线，沿线国家受SWIFT制裁风险高，替代需求更强，是CIPS增长的结构性支撑。

7.区域6.监管
去中心化加密货币和美国过度使用SWIFT制裁工具对SWIFT网络造成不利影响，俄罗斯和中国都在努力建立替代网络；SWIFT同时在探索支持加密数字货币。Wikipedia / 人人都是产品经理交叉
CIPS崛起的外部动因：①地缘政治武器化推动去美元化；②加密货币提供技术性替代路径。SWIFT正通过探索加密货币来维护生态位。
SWIFT竞争去美元化加密货币

中国于1983年加入SWIFT，中国4大国有商业银行均为SWIFT活跃成员；中国在SWIFT董事会只有1个席位，美国、比利时等各有2席，CEO由欧洲人担任。人人都是产品经理 / SWIFT治理结构
SWIFT治理结构中国话语权有限（1/25席），这是中国发展CIPS替代路径的深层动机——避免依赖西方主导的金融通信基础设施。

6.监管4.竞争
CLS（持续连接结算）每天结算金额数万亿美元，占全球货币日交易总价值90%以上，通过PvP（券款对付）解决赫斯塔特风险。人人都是产品经理 / 清算体系
CLS是全球外汇结算最重要基础设施，人民币是否纳入CLS是人民币国际化深化的重要衡量指标，目前仍主要通过双边安排解决。
